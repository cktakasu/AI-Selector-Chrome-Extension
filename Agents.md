# AI Selector プロジェクト方針

## 1. 開発の軸
本プロジェクトは **Google Chrome 拡張機能** をメインのターゲットとして開発されています。

## 2. ブラウザ対応状況
Chrome を中心としつつ、コードベースの共通化により、以下のブラウザでも同様に動作する「クロスプラットフォーム設計」を採用しています。

- **Google Chrome** (メイン・ターゲット)
- **Microsoft Edge** (Chromiumベースとしての完全互換)
- **Apple Safari** (Safari Web Extension 形式への自動同期・互換レイヤー対応)

## 3. 技術的アプローチ
- `lib/browser.ts` によるブラウザAPIの抽象化
- ポインターイベントによるマルチデバイス・マルチブラウザ対応の操作系
- Viteビルド成果物の自動同期パイプラインによる、Safariプロジェクトへの最新コード供給
